# 🛡️ AI Ops Sentinel - Security Monitoring Dashboard

**SentinelAI** is an advanced security ecosystem built to protect web applications against modern cyber threats. This repository contains the **Operations Dashboard**, a "Silent Luxury" styled interface designed to visualize and manage threats detected by an intelligent WAF engine in real-time.

## ✨ Key Features

* **⚡ Real-Time Threat Detection:** Connected to a MySQL backend that logs SQL Injection attempts and unauthorized intrusions instantly.
* **🔒 Bank-Level Privacy:** Implemented **AES-256-CBC** encryption to protect sensitive user metadata, ensuring high security standards.
* **🤖 AI-Driven Architecture:** Designed for automated risk scoring and heuristic analysis of incoming traffic patterns.
* **💎 Premium UX/UI:** A minimalist, sophisticated dark-mode interface built with **Tailwind CSS**, optimized for professional Security Operations Centers (SOC).

## 🛠️ Tech Stack

* **Backend:** PHP 8.x (utilizing PDO for secure database interactions).
* **Database:** MySQL (Relational data management).
* **Frontend:** Tailwind CSS & HTML5.
* **Security:** OpenSSL for data encryption and WAF logic integration.

## 🧠 The AI Component

In this project, AI acts as the "Central Brain" of the operation:
1.  **AI Risk Scoring:** Automatically evaluates the danger level of the source IP and the request payload.
2.  **Heuristic Monitoring:** Detects suspicious behavioral patterns that traditional, rule-based firewalls might miss.
3.  **Human-Readable Insights:** Translates complex technical logs into executive summaries, allowing for faster decision-making.
